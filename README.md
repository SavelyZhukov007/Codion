# Codion — Портфолио команды

Сайт-визитка команды разработчиков **Codion** (КИПФИН, Финансовый Университет).

## Структура проекта

```
codion/
├── index.html          ← главная страница
├── css/
│   └── style.css       ← скомпилированные стили
├── scss/
│   └── style.scss      ← исходный SCSS (редактируйте его)
├── js/
│   └── main.js         ← JavaScript
├── images/             ← все изображения сайта
├── insert.py           ← GUI-менеджер контента (Python + Tkinter)
└── README.md
```

## Как запустить

Просто откройте `index.html` в браузере.

## Редактирование контента

### Через insert.py (рекомендуется)

```bash
# Установите зависимости
pip install tkinterweb Pillow

# Запустите из корневой папки сайта
python insert.py
```

Приложение позволяет:
- Добавлять новые **проекты** с изображением и технологиями
- Добавлять **новости и достижения**
- Добавлять **участников команды** с фото
- Просматривать **превью сайта** прямо в приложении
- Видеть список всех добавленных блоков

### Вручную

Откройте `index.html` и отредактируйте нужные секции.  
Маркеры `<!-- DYNAMIC_BLOCKS_START` и `<!-- DYNAMIC_BLOCKS_END` указывают зону для динамических блоков.

## Как пересобрать SCSS

```bash
pip install libsass
python -c "import sass; open('css/style.css','w').write(sass.compile(filename='scss/style.scss'))"
```
